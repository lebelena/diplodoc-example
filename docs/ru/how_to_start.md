# С чего начать?

Первое, что отображается при входе в личный кабинет — страница мероприятий. 

![_screen_start_01](_images/_screen_start_01.png){.center width=1200}

Вы можете:
* ознакомиться с доступными для вас мероприятиями;
* создать новое мероприятие:
  * добавить [нового контрагента](./how_to_counterparty.md);
  * создать [заявку для склада](./how_to_task.md);

Также рекомендуется проверить наполненность справочников: номенклатура, пространства, контрагенты — и при необходимости добавить новые элементы. 

Если требуется сделать что-то конкретное, обратитесь к соответствующему разделу в практическом руководстве.
