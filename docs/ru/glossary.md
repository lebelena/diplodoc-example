## HR

**H**

{% list tabs accordion %}

- HR-бренд

  восприятие компании как работодателя на рынке труда

{% endlist %}

**K**

{% list tabs accordion %}

- Key Performance Indicator (KPI)

  ключевые показатели эффективности, которые измеряют, насколько успешно компания или отдельный сотрудник достигает поставленных целей. Они помогают оценить эффективность работы, анализировать результаты, планировать задачи и мотивировать персонал путем определения измеримых целей

{% endlist %}

**W**

{% list tabs accordion %}

- Welcome-тренинг

  тренинг, предназначенный для новых сотрудников, целью которого является знакомство с историей, миссией, ценностями, структурой компании и основными правилами внутреннего распорядка

{% endlist %}

**Б**

{% list tabs accordion %}

- Больничный лист

  документ, подтверждающий временную нетрудоспособность сотрудника

{% endlist %}

**В**

{% list tabs accordion %}

- Вакансия

  должность, на которую идёт поиск кандидатов

- Вовлеченность

  эмоциональная заинтересованность сотрудника в своей работе, ценностях и целях компании, выражающаяся в активном участии и инициативности

{% endlist %}

**Г**

{% list tabs accordion %}

- Грейд

  уровень квалификации сотрудника в структуре компании, определяющий его компетенции, зону ответственности и уровень вознаграждения

{% endlist %}

**З**

{% list tabs accordion %}

- Заработная плата

  вознаграждение, выплачиваемое работнику за выполненную работу, включающее основную часть (оклад), компенсационные выплаты (за условия труда) и стимулирующие выплаты (премии)

{% endlist %}

**И**

{% list tabs accordion %}

- Индивидуальный план развития (ИПР)

  персонализированный маршрут развития сотрудника, включающий обучающие мероприятия и направленный на рост профессиональных компетенций

- Испытательный срок

  период, в течение которого оценивается соответствие нового сотрудника занимаемой должности

{% endlist %}

**К**

{% list tabs accordion %}

- Кадровый резерв

  группа сотрудников, которые имеют потенциал для развития и могут занять ключевые должности в будущем

- Кандидат

  лицо, претендующее на вакантную должность

- Карьерный трек

  последовательность должностей и ролей внутри компании, по которой сотрудник может продвигаться, развивая свои навыки и компетенции

- Компетенции

  совокупность знаний, навыков, личностных качеств и опыта, необходимых для успешного выполнения профессиональных задач

- Корпоративная культура

  система ценностей, норм и правил, разделяемых сотрудниками компании и определяющих стиль взаимодействия внутри организации

{% endlist %}

**Л**

{% list tabs accordion %}

- Лояльность

  приверженность сотрудника компании, выражающаяся в готовности оставаться в ней, поддерживать её интересы и активно участвовать в её развитии

{% endlist %}

**М**

{% list tabs accordion %}

- Мотивация

  побуждение к действию, динамический психофизиологический процесс, который определяет направленность, активность и устойчивость поведения человека, позволяя ему удовлетворять свои потребности и достигать целей

{% endlist %}

**О**

{% list tabs accordion %}

- Обратная связь (ОС)

  информация о результатах деятельности сотрудника, предоставляемая ему с целью улучшения эффективности и профессионального развития

- Оклад

  фиксированная часть вознаграждения за труд

- Онбординг

  процесс интеграции нового сотрудника в компанию

- Опрос

  инструмент сбора информации от сотрудников по заданной теме

- Отпуск

  время отдыха, предоставляемое сотруднику с сохранением места работы, должности и среднего заработка

- Офбординг

  процесс официального завершения трудовых отношений с сотрудником, включающий передачу дел, выходные интервью и оформление документов

{% endlist %}

**П**

{% list tabs accordion %}

- Паспортные данные (ПД)

  персональные данные, содержащие сведения, указанные в паспорте гражданина, которые позволяют однозначно идентифицировать личность

- Премия

  переменная часть вознаграждения, выплачиваемая за достижение определённых результатов

{% endlist %}

**Р**

{% list tabs accordion %}

- Резюме

  документ, содержащий информацию об образовании, опыте, навыках и личных сведениях кандидата

- Ротация

  временное или постоянное перемещение сотрудника на другую должность внутри компании с целью развития, предотвращения выгорания или оптимизации кадрового состава

{% endlist %}

**С**

{% list tabs accordion %}

- Собеседование

  целенаправленная беседа представителей компании с кандидатом для взаимной оценки соответствия вакансии и ожиданий

- Сотрудник

  физическое лицо, которое работает по трудовому договору в организации и получает за это заработную плату

{% endlist %}

**Т**

{% list tabs accordion %}

- Тестирование

  инструмент оценки знаний, навыков, способностей, личностных качеств или профессиональной пригодности сотрудника или кандидата с помощью стандартизированных заданий, вопросов или практических кейсов

- Тренинг

  краткосрочный процесс обучения конкретным навыкам или знаниям

- Трудовой договор

  документ, регулирующий трудовые отношения между сотрудником и компанией

- Трудовой кодекс Российской Федерации (ТК РФ)

  основной законодательный акт, регулирующий трудовые отношения между работником и работодателем, закрепляющий их взаимные права и обязанности

{% endlist %}

**У**

{% list tabs accordion %}

- Увольнение

  прекращение трудовых отношений между сотрудником и компанией

{% endlist %}

**Ш**

{% list tabs accordion %}

- Штатное расписание

  документ, определяющий структуру, численность и должностной состав компании

{% endlist %}

## IT и Аналитика

**A**

{% list tabs accordion %}

- A/B тестирование

  метод сравнения двух версий продукта или страницы (A и B) для определения, какая из них работает лучше

- Application Programming Interface (API)

  набор правил и протоколов, который позволяет разным программам взаимодействовать друг с другом и обмениваться данными. Это как «договор» между двумя программами, где один сервис отправляет запрос другому и получает ответ, чтобы выполнить задачу или получить информацию

{% endlist %}

**B**

{% list tabs accordion %}

- Business Intelligence (BI)

  технологии и стратегии для анализа бизнес-информации и предоставления удобных для восприятия отчетов и дашбордов

{% endlist %}

**C**

{% list tabs accordion %}

- COM-объекты (Component Object Model)

  бинарные программные компоненты, созданные с использованием технологии Microsoft Component Object Model, которая позволяет различным программам взаимодействовать друг с другом независимо от языка программирования или платформы

- Continuous Integration / Continuous Delivery (CI/CD)

  методология автоматизации процессов разработки программного обеспечения, включающая частую интеграцию кода, автоматизированное тестирование и развертывание

{% endlist %}

**D**

{% list tabs accordion %}

- Data Lake

  хранилище большого объема сырых данных в их первоначальном формате

- Data Science

  междисциплинарная область, использующая научные методы, процессы, алгоритмы и системы для извлечения знаний и инсайтов из структурированных и неструктурированных данных

- Data Warehouse

  централизованное хранилище данных, предназначенное для отчетности и анализа

- DevOps (Development, Operations)

  культура и набор практик, объединяющих разработку и IT-эксплуатацию, чтобы сократить время выхода продукта и улучшить его качество

{% endlist %}

**E**

{% list tabs accordion %}

- ETL (Extract, Transform, Load)

  процесс извлечения данных из различных источников, их преобразования в пригодный для анализа вид и загрузки в целевую систему (например, хранилище данных)

{% endlist %}

**L**

{% list tabs accordion %}

- LCP (LogiCore Platform)

  это интеграционная цифровая платформа, предназначенная для управления всеми операционными процессами предприятия (логистическими, складскими, мерчандайзинговыми и финансовыми) на единой технологической основе. Система построена по методологии Control Tower и объединяет три ключевых подсистемы (SLC, TIC, PLC)

{% endlist %}

**M**

{% list tabs accordion %}

- Machine Learning (ML)

  раздел искусственного интеллекта, изучающий алгоритмы, которые позволяют компьютерам обучаться на данных без явного программирования

- MiniApp (МиниАпп)

  программное обеспечение, с помощью которого исполнители могут получать задачи, отчитываться за выполненные работы, анализировать результативность и выгружать выставленные счета

- Minimum Viable Product (MVP)

  пминимально жизнеспособный продукт для тестирования гипотез, первая стадия готового продукта

{% endlist %}

**P**

{% list tabs accordion %}

- Product Owner

  роль в Scrum, ответственная за максимизацию ценности продукта, управление бэклогом и определение целей

{% endlist %}

**S**

{% list tabs accordion %}

- SQL (Structured Query Language)

  это язык структурированных запросов, используемый для управления данными в реляционных базах данных. С помощью SQL можно создавать, изменять, удалять и извлекать данные, а также управлять их структурой

{% endlist %}

**U**

{% list tabs accordion %}

- User Story

  неформальное, краткое описание функции продукта, написанное с точки зрения конечного пользователя

{% endlist %}

**W**

{% list tabs accordion %}

- Waterfall

  последовательная модель разработки, где каждая фаза (сбор требований, дизайн, реализация, тестирование) завершается до начала следующей

{% endlist %}

**А**

{% list tabs accordion %}

- Аджайл (от англ. Agile)

  гибкий подход к управлению проектами, основанный на итерациях

{% endlist %}

**Б**

{% list tabs accordion %}

- Блокер (от англ. Blocker)

  проблема, из-за которой выполнение задачи невозможно или существенно затруднено

- Бэкенд (от англ. Backend)

  серверная часть приложения, невидимая пользователю, отвечающая за бизнес-логику и обработку данных

- Бэклог (от англ. Backlog)

  приоритизированный список задач проекта

{% endlist %}

**Д**

{% list tabs accordion %}

- Дашборд (от англ. Dashboard)

  интерактивная панель с ключевыми метриками, графиками и отчетами

- Дедлайн (от англ. Deadline)

  крайний срок выполнения задачи, проекта или другого действия

- Деплой (от англ. Deploy)

  размещение готовой версии программного обеспечения на платформе, доступной пользователям

- Дока (от англ. Dock)

  техническая или пользовательская документация

{% endlist %}

**К**

{% list tabs accordion %}

- Кибербезопасность

  практика защиты систем, сетей и программ от цифровых атак

- Контейнеризация (от англ. Docker)

  технология упаковки приложения и всех его зависимостей в изолированный контейнер, который может работать на любой совместимой системе

{% endlist %}

**М**

{% list tabs accordion %}

- Метрика

  количественный показатель, используемый для оценки эффективности

- Микросервисы (от англ. Microservices)

  архитектурный стиль, при котором приложение состоит из небольших, независимо развертываемых сервисов

{% endlist %}

**П**

{% list tabs accordion %}

- Прогнозирование

  предсказание будущих значений на основе исторических данных

{% endlist %}

**Р**

{% list tabs accordion %}

- Релевантность данных

  степень соответствия данных поставленной аналитической задаче

- Релиз (от англ. Release)

  публикация готовой версии продукта

- Ретро (от англ. Retrospective)

  встреча для анализа завершенного спринта

{% endlist %}

**С**

{% list tabs accordion %}

- Сводная таблица

  инструмент для агрегации и анализа данных с группировкой по параметрам

- Сделка (ERP 1C)

  название поручения в системе ERP 1C

- Сезонность

  повторяющиеся изменения данных, связанные с временем года, праздниками и другими факторами

- Скрам (от англ. Scrum)

  agile-методика с фиксированными итерациями (спринтами)

- Смарт-процесс

  название заказа в системе ERP 1C

- Спринт (от англ. Sprint)

  фиксированный период разработки, обычно 1–4 недели

{% endlist %}

**Т**

{% list tabs accordion %}

- Техдолг (от англ. Tech Debt)

  накопленные проблемы в коде, требующие исправления

{% endlist %}

**Ф**

{% list tabs accordion %}

- Фича (от англ. Feature)

  функциональность или особенность продукта

- Фронтенд (от англ. Frontend)

  клиентская часть приложения (интерфейс)

{% endlist %}

## Делопроизводство

**В**

{% list tabs accordion %}

- Возврат документов

  передача заказчику или в офис оригиналов документов после завершения операций

{% endlist %}

**Д**

{% list tabs accordion %}

- Договор

  документ, подтверждающий соглашение между двумя (или более) сторонами, устанавливающее, изменяющее или прекращающее взаимные права и обязанности подписантов

{% endlist %}

**И**

{% list tabs accordion %}

- Индивидуальный предприниматель (ИП)

  это физическое лицо, которое зарегистрировано в установленном законом порядке для ведения предпринимательской деятельности без создания юридического лица. Это означает, что один человек самостоятельно занимается бизнесом, направленным на получение прибыли, и при этом несёт полную ответственность по своим обязательствам всем своим имуществом

{% endlist %}

**О**

{% list tabs accordion %}

- Общество с ограниченной ответственностью (ООО)

  это одна из самых распространенных форм юридического лица для ведения бизнеса. Участники такого общества не несут ответственности по его долгам всем своим личным имуществом, а только в пределах вкладов в уставный капитал

- Оформление документов

  подготовка и заполнение сопроводительных документов (накладные, акты, приёмосдаточные ведомости)

{% endlist %}

**С**

{% list tabs accordion %}

- Самозанятый (СЗ)

  физическое лицо, которое занимается предпринимательской деятельностью без создания юридического лица и без регистрации в налоговых органах в качестве ИП

{% endlist %}

**П**

{% list tabs accordion %}

- Поручение (LCP)

  высокоуровневое указание от клиента на выполнение услуг. Документ, который инициирует безналичный перевод средств со счета плательщика на счет получателя за оказанные работы (монтаж, сборка и установка и др.); содержит необходимые реквизиты для перевода, создаётся только на основе договора. Может содержать несколько заказов. Детализация выполненных работ контрагента за отчетный период с 13 по 28 и с 28 по 13 число каждого месяца. Поручение накладывает ограничение на возможные услуги в заказе

{% endlist %}

**Ч**

{% list tabs accordion %}

- Чек СЗ

  это основной отчётный документ для самозанятых, подтверждающий факт получения дохода и уплату налога

{% endlist %}

## Менеджмент

**C**

{% list tabs accordion %}

- Control Tower

  это подход к управлению логистикой, который объединяет данные из разных источников в едином центре для обеспечения прозрачности, контроля и оперативного реагирования в режиме реального времени. Эта методология использует технологические платформы для сбора данных, аналитики и взаимодействия между всеми участниками цепи поставок для принятия более обоснованных решений и оптимизации процессов

{% endlist %}

**А**

{% list tabs accordion %}

- Адресная программа (АП)

  реестр адресов, в которых предполагается проведение работ в рамках одного заказа.

- Алгоритм работы отдела

  четко определенная последовательность действий, разделенных на циклы проекта, имеющих срок реализации, исполнителя и участников с их ролью согласно функциональной матрице ответственности

{% endlist %}

**В**

{% list tabs accordion %}

- Вид оборудования

  внутренняя сущность в системе, описывающая вид объекта, над которым выполняются операции и производится работа. Характеристики: габариты, вес, привязка к операциям, ограничения, количество персонала. Назначение: базовая единица учёта и планирования ресурсов

{% endlist %}

**Д**

{% list tabs accordion %}

- Деловые Линии

  транспортная компания, которая по договору выполняет услуги перевозки

- Директор магазина (ДМ)

  руководитель конкретной торговой точки, отвечающий за ее операционную деятельность, включая приемку товара, мерчандайзинг, выполнение плана продаж

- Дополнительное место продаж (ДМП)

  в контексте розничной торговли, это специальное место в торговом зале, где товар, представленный в основном ассортименте, выкладывается отдельно, с целью привлечения внимания покупателей и увеличения продаж

{% endlist %}

**З**

{% list tabs accordion %}

- Задача

  совокупность операций, выполняемых в конкретном пространстве, в конкретное время, конкретным контрагентом, над конкретными оборудованиями, является частью заказа

- Заказ (в СУУ)

  совокупность услуг, которые необходимо выполнить в определенные сроки за рассчитанный бюджет, сформированная в адрес контрагента Паблик Тотем и реализуемая агентом ПОСМ.РЦ

- Заказ (в LCP)

  формализованный документ в системе, описывающий конкретный объём работ и услуг, подлежащих исполнению, является частью поручения

- Занесение

  состояние процесса, является начальным и означает создание процесса клиентским менеджером

- Запрос

  состояние процесса, означает поступление процесса на расчет от клиентского менеджера

{% endlist %}

**К**

{% list tabs accordion %}

- Калькуляция

  состояние процесса, говорит о том, что процесс рассчитан и ожидает подтверждения от КМ

- Канбан (от англ. Kanban)

  метод управления рабочими процессами, который использует визуализацию задач на канбан-доске, который подразумевает ограничение незавершенной работы и постоянные улучшения для повышения эффективности команды и прозрачности процесса. Цель канбан системы – это обеспечить непрерывный поток работы, помочь команде начать и закончить задачи, а также выявить узкие места в процессе

- Клиентский менеджер (КМ, эккаунт)

  менеджер, взаимодействующий с представителем клиента в рамках Заказа, получающий необходимую информацию о нем и передающий ее агенту. Является буфером между Заказчиком и Исполнителем

- Контрагент

  деловой партнёр, с которым компания определяет взаимные права и обязанности на основании Договора. Официальное юридическое или физическое лицо (например, ИП), которое фигурирует в регламентированном учёте с полными официальными данными. Связь между ними не всегда прямая: один партнёр может включать несколько контрагентов (например, для одной сети магазинов), а один контрагент может быть связан с несколькими партнёрамим

- Корреспонденция

  адрес контрагента для забора документов и прочих поручений

{% endlist %}

**Л**

{% list tabs accordion %}

- Лицо принимающее решение (ЛПР)

  любое физическое или юридическое лицо, обладающее полномочиями для принятия решения хода проекта – представитель клиента, заказчика, исполнителя, сотрудник торговой точки

- Личный кабинет (ЛК)

  специализированный раздел цифровой платформы, предназначенный для предоставления пользователям персонализированного интерфейса и функционала

{% endlist %}

**М**

{% list tabs accordion %}

- Маржинальность

  отношение себестоимости к выручке, выраженное в процентах

- Менеджер проекта (МП)

  специалист, который отвечает за планирование, организацию и реализацию проекта от получения Заказа до выставления счетов клиенту. Он руководит командой, управляет ресурсами, бюджетом и сроками, а также координирует коммуникации со всеми участниками, чтобы достичь целей проекта в установленных рамках

- Мероприятие

  последовательность зафиксированных событий в системе

- Мониторинг показателей

  процесс отслеживания метрик

{% endlist %}

**О**

{% list tabs accordion %}

- Обеспечение деятельности

  группа операций, которая необходима для проведения работ в пространстве в рамках услуг доставки, монтажа, утилизации, аудита, мерчандайзинга и ремонта

- Оператор персональных данных

  государственный орган, муниципальный орган, юридическое или физическое лицо, которое организует и (или) осуществляет обработку персональных данных, а также определяет цели, состав и содержание их обработки

- Операция

  единица работы, имеющая свою тарификацию, является частью задачи

- Отгрузка

  состояние процесса, говорит о присвоении конечных статусов всем задачам процесса

- Отклонение

  разница между плановым и фактическим значением показателя

{% endlist %}

**П**

{% list tabs accordion %}

- Партнёр (в ERP 1С)

  управленческое понятие, охватывающее все деловые отношения с группой компаний или организацией

- Подтверждение

  состояние процесса, говорит о согласии клиентского менеджера на сумму расчета процесса

- Пользователь

  зарегистрированное на Платформе и получившее доступ к Личному кабинету лицо. Если Пользователь не является полностью дееспособным, регистрация осуществляется его родителем или иным законным представителем; все действия, осуществляемые таким Пользователем на Платформе, совершаются с согласия его родителя или иного законного представителя. Если Пользователь является полностью дееспособным (достигшим 18 лет или объявленным полностью дееспособным в соответствии со ст. 27 Гражданского кодекса Российской Федерации), регистрация и все действия, совершаемые на Платформе, осуществляются Пользователем самостоятельно

- Правообладатель

  субъект права (физическое или юридическое лицо), который управляет цифровой платформой и владеет правами на ее программное обеспечение, аппаратное обеспечение и другую связанную с ней интеллектуальную собственность

- Префикс (LCP)

  уникальная аббревиатура контрагента, заказчика или исполнителя

- Приказ

  управленческий триггер, в котором группируются задачи по определенным критериям, из которого запускаются мероприятия

- Принятие подтверждения (ФК1)

  состояние процесса, говорит о взятии процесса в работу менеджером проекта

- Процесс (СУУ)

  сущность, олицетворяющая отдельные услуги в рамках заказа

{% endlist %}

**Р**

{% list tabs accordion %}

- Распределительный центр (РЦ)

  хаб/склад/город базирования нашего контрагента, из которого впоследствии происходит реализация проектов в рамках этого города и в городах сателлитах

- Роадмэп (от англ. Roadmap)

  дорожная карта развития продукта – стратегический, часто визуальный, план, который описывает ключевые этапы, сроки, задачи и цели для достижения общей цели в проекте, компании или развитии продукта

{% endlist %}

**С**

{% list tabs accordion %}

- Сервис Платформы

  комплексная автоматизированная система функций, обеспечивающая полный цикл организации и управления транспортными операциями, складскими процессами и выполнением полевых работ. Включает инструменты для формирования документации, планирования перевозок, выбора исполнителей, контроля перемещений, обработки и учета складской продукции, координации монтажных и сервисных мероприятий, сбора доказательств выполнения задач

- Склад контрагента (СК)

  пространство, арендуемое для хранения отправленного оборудования в регионах

- Событие (LCP)

  последовательность зафиксированных действий или этапов бизнес-процесса, отражающих ход выполнения мероприятия в системе. Каждое событие содержит: название действия, исполнителя/автора (конкретное ответственное лицо), момент времени начала и окончания

- Совет директоров

  орган управления в компании, который отвечает за стратегическое развитие и контроль над деятельностью руководителей подразделений (Скрипник Дмитрий Алексеевич, Элиашвили Александр Абрамович)

- Согласование работ

  предварительная договоренность с заказчиком или внутренними службами объема, сроков и условий выполнения операций

- Старт план услуга

  день, в который надо начинать производить работы по процессам/задачам

{% endlist %}

**Т**

{% list tabs accordion %}

- Тип оборудования

  внутренняя сущность в системе, объединяющая оборудование разных видов в конкретные группы на основе определенных общих признаков

{% endlist %}

**У**

{% list tabs accordion %}

- Услуга

  комплекс действий, произведенный исполнителем услуги по заказу покупателя (потребителя услуги), имеющий целью удовлетворение потребности покупателя (потребителя услуги)

- Учетная запись

  хранимая на Платформе совокупность данных о Пользователе, необходимая для его опознавания (аутентификации) и предоставления доступа к его персональным данным и настройкам на Платформе

{% endlist %}

**Ф**

{% list tabs accordion %}

- Факт вход в РЦ

  факт получения груза контрагентом

- Фин. контроль (ФК2)

  состояние процесса, говорит о выставлении клиентским менеджером счета клиенту

- Финиш план услуга

  день завершения работ по процессам/заказам

{% endlist %}

**Ц**

{% list tabs accordion %}

- Целевое значение

  плановый показатель, который необходимо достичь

{% endlist %}

**Э**

{% list tabs accordion %}

- Эффективность

  достижение максимального результата при минимальных затратах ресурсов, таких как время, деньги или усилия. Это соотношение между полученным результатом (эффектом) и затратами, которые были для этого использованы. Таким образом, эффективность означает делать что-то правильно, с наименьшими потерями и максимальной отдачей

{% endlist %}

## Сервисная логистика

**P**

{% list tabs accordion %}

- POSM Lifecycle Core (PLC)

  ядро (система) жизненного цикла POSM

- Point of Sale Materials (POSM/ПОСМ)

  РИМ (Рекламно-информационные материалы) / РТО (Рекламно-торговое оборудование), размещаемые в точках продаж для привлечения внимания покупателей к конкретным товарам или услугам

{% endlist %}

**А**

{% list tabs accordion %}

- Аудит

  комплекс проверочных мероприятий: измерения, оценка состояния оборудования и объекта, выявление дефектов, фотофиксация для отчётности

{% endlist %}

**Б**

{% list tabs accordion %}

- Баннер (S-XL)

  вид оборудования, отрезок материала произвольного размера (тканевого, винилового, синтетического) с нанесенным на поверхность рекламным слоганом и изображением определенного товара, размещаемый внутри магазина или на входе

- Бренд-зона (S-M)

  тип оборудования, группа сложносоставных изделий, с возможностью подключения к сети в бренд-зоне M

- Бренд-зона / Shop in Shop / ШиШ

  специально выделенное и оформленное пространство внутри магазина, предназначенное для продвижения конкретного бренда или группы товаров

{% endlist %}

**В**

{% list tabs accordion %}

- Вобблер

  вид оборудования, «приманки» на ленте, на которые ловят внимание покупателей. Двустороннее изображение из картона или плотной бумаги на гибкой ножке

- Выкладка

  размещение продукции определенного бренда в торговом зале в заранее определенном месте на заранее определенном оборудовании согласно планограмме

{% endlist %}

**Г**

{% list tabs accordion %}

- Геометка

  данные о географическом местоположении и дате и/или времени, привязанные к фотографии

- Гипермаркет (ГМ)

  крупноформатная торговая точка с широким ассортиментом продуктов и непродовольственных товаров

- Гондолла

  островной стеллаж, на полках которого свободно располагаются товары малого и среднего размеров

{% endlist %}

**Д**

{% list tabs accordion %}

- Демонтаж

  разборка и снятие ранее установленного оборудования

- Дисплей картонный

  тип оборудования, простосоставное изделие из картона, интегрируемое в оборудование ТТ

- Дисплей/Глорифайер

  простосоставное изделие из металла и/или пластика, интегрируемое в оборудование ТТ, с возможным подключением к сети

{% endlist %}

**К**

{% list tabs accordion %}

- Картонный стенд

  рекламный стенд различной формы, сделанный из гофрокартона, с нанесением полноцветного изображения

- Клининг

  уборка и очистка объекта после проведения работ (монтажа, ремонта, демонтажа)

- Код Бордовый

  условное обозначение для ЧП или ситуации, требующей немедленного реагирования всех вовлеченных сторон

- Контрагент / техник / исполнитель/ «последние руки»

  юридическое лицо, с которым заключен договор на выполнение работ/услуг в цепочке поставок и/или технического мерчандайзинга

- Кристалайт/лайтбокс

  световая панель из акрилового стекла (двухстороння или односторонняя), со сменными имиджами

{% endlist %}

**М**

{% list tabs accordion %}

- Магнитный винил

  рекламное изображение, пластина, при изготовлении которой используются полимер и ферриты, являющиеся магнитным наполнителем. На оборотной стороне такой пластины, как правило, клеевой слой

- Маршрут

  совокупность задач на работу в одном заказе на 1 день

- Монтаж оборудования

  сборка, установка и подключение оборудования на объекте

{% endlist %}

**Н**

{% list tabs accordion %}

- Накопители

  тип оборудования, конструкция оригинального дизайна из пластика или металла с нанесенной рекламной информацией, представляющая собой емкость, стойку или панель с подставками или направляющими, предназначенная для хранения, демонстрации и поштучной подачи товара покупателю в местах продаж

- Наряд

  совокупность задач и маршрутов на один рабочий день для конкретного исполнителя

- Настройка оборудования

  регулировка данного оборудования для приведения его в рабочее состояние

{% endlist %}

**О**

{% list tabs accordion %}

- Островной стенд

  стенд устанавливаемый автономно от прочих рекламных материалов

- Оформление стеллажа

  оформление от 2-10 секций стандартного РТО

{% endlist %}

**П**

{% list tabs accordion %}

- Панели

  декоративно-функциональное оборудование, обычно прямоугольной формы, крепящееся к стене, полу

- Паразиты

  тип оборудования, POS материалы, навешивающиеся к уже находящемуся в торговой зоне оборудованию. В основном в прикассовой зоне, иногда на стеллажах

- Перепробег

  расстояние от города РЦ до места проведения работ

- Подвесное оборудование

  тип оборудования, представляет собой конструкцию рекламно-информационного характера, которую размещают под потолком, на стенах

- Пушеры

  толкатели для шелф-систем, продвигающие товар к переднему краю прилавка

{% endlist %}

**Р**

{% list tabs accordion %}

- Рабочий день

  совокупность операций на день в 1 пространстве

{% endlist %}

**С**

{% list tabs accordion %}

- Сменные имиджи

  тип оборудования, сменные POS материалы, устанавливаемые в оборудование типа лайтбокс, кристалайт, таблички, стикеры, винилы, баннеры

- Спецмонтаж

  установка оборудования, требующая участия специалистов (высотные работы, электромонтаж, сложные технические подключения)

- Стеллажи/стенды

  тип оборудования, сложносоставное изделие, предназначенное для размещения на нем товара, выделения и продвижения товара определенной торговой марки/брэнда

- Стенд торцевой

  стенд, устанавливаемый на торце стационарного оборудования в ТТ, обычно крепящийся посредством струбцин

- Стикер

  самоклеящейся пленка для размещения на полу

- Страйплента/стрипхолдер/стриплента

  вид оборудования, полоса из пластика или полиэтилена, на которую крепится легкий и небольшой товар

- Супермаркет (СМ)

  магазин самообслуживания среднего формата с преимущественно продовольственным ассортиментом, расположенный в жилых районах или ТЦ

{% endlist %}

**Т**

{% list tabs accordion %}

- Табличка

  информационная пластина из различных материалов, обычно прямоугольного формата

- Театрализация

  тематические выкладки с эффектным дизайном, обычно с большой многоярусной выкладкой товара

- Технический мерчандайзинг

  комплекс услуг, направленный на повышение продаж в отдельных розничных точках (производство и обслуживание РИМ/РТО, логистические услуги, установка РИМ/РТО, выкладка продукции, диагностика и исправление неполадок РТО, аудиты технические)

- ТЗ/спецификация

  это информация, в которой заказчик прописывает требования и условия выполнения задачи, ожидаемый результат и объяснение, как эту задачу выполнять

- Торговая точка (ТТ)

  конкретный магазин, павильон, киоск или иное место розничной продажи товаров

{% endlist %}

**Ф**

{% list tabs accordion %}

- Фотофиксация

  фотографирование выполненных работ для подтверждения его состояния, комплектации и факта выполнения работ

- Фриз

  декоративные элементы, чаще в виде длинной узкой панели, баннера или короба, тянущиеся вдоль наружной стены

{% endlist %}

**Х**

{% list tabs accordion %}

  выезд исполнителя на объект без выполнения запланированных работ (например, из– за отсутствия доступа или оборудования)

{% endlist %}

**Ц**

{% list tabs accordion %}

- Ценниковыделитель

  POSM с рекламными полями и окном для вставки ценника. Назначение – привлечь внимание потребителя к определенной продукции

{% endlist %}

**Ш**

{% list tabs accordion %}

- Шелфбаннер (ШБ, от англ. Shelf Banner)

  закреплённый к полке разделитель для товаров разной категории

- Шелфпак

  вид оборудования, рекламная конструкция из картона с брендированным торцом, предназначенная для обозначения продуктов определенной торговой марки на полках

- Шелфтокер (ШТ, от англ. Shelf Talker)

  вид оборудования, представляет собой небольшую полоску с мотивирующим изображением (логотипом, названием бренда), прикрепленной к полке с товаром

{% endlist %}

## Склады

**E**

{% list tabs accordion %}

- ERP (Enterprise Resource Planning)

  система планирования ресурсов предприятия, комплексная программа для автоматизации и управления основными бизнес-процессами компании (финансы, закупки, производство, продажи, персонал)

{% endlist %}

**F**

{% list tabs accordion %}

- FIFO (First In, First Out)

  метод управления запасами, при котором первыми отпускаются товары, поступившие на склад раньше

{% endlist %}

**L**

{% list tabs accordion %}

- LIFO (Last In, First Out)

  метод управления запасами, при котором первыми отпускаются товары, поступившие на склад последними

{% endlist %}

**S**

{% list tabs accordion %}

- SKU (Stock Keeping Unit, СКЮ)

  уникальный код, присваиваемый каждому материалу для его учета и идентификации

- Space Lifecycle Core (SLC)

ядро жизненного цикла пространства - система управления пространствами присутствия активов (материалы, товары, срок эксплуатации, качество использования актива) распространяется за пределы привычного понимания WMS т.к. учитывает не только классические принципы складского учета, но и позволяет контролировать пространства: Торговых сетей, торгового оборудования с размещенным на них продуктом. Позволяет контролировать цифровые транзакции движения объектов сервиса

{% endlist %}

**А**

{% list tabs accordion %}

- Адресное хранение

  система организации склада, при которой каждому месту хранения (стеллажу, полке, ячейке) присваивается уникальный адрес

{% endlist %}

**Б**

{% list tabs accordion %}

- Брак

  оборудование, поступившее на склад, не бывшее в эксплуатации, но имеющее повреждения, возникшие в процессе производства или доставки с центрального склада

{% endlist %}

**В**

{% list tabs accordion %}

- Выборочная инвентаризация

  инвентаризация только части имущества (например, определенные товарные позиции, основные средства или деньги), а не всех активов компании

{% endlist %}

**Д**

{% list tabs accordion %}

- Дабл-стек (от англ. Double stack)

  метод размещения одного загруженного поддона поверх другого

{% endlist %}

**З**

{% list tabs accordion %}

- Закупка

  приобретение необходимых материалов или комплектующих для выполнения работ

- Заявка на отгрузку Москва

  документ, на основании которого происходит выдача/прием материалов на склад

- Заявка на склад

  инструмент, необходимый для формирования драфта документа списания оборудования со склада или поступления оборудования на склад

{% endlist %}

**И**

{% list tabs accordion %}

- Инвентаризация

  проверка фактического наличия и состояния имущества организации и её обязательств путем сверки с данными бухгалтерского учёта

{% endlist %}

**К**

{% list tabs accordion %}

- Киттинг

  процесс комплектации товаров на складе для формирования единого готового набора или комплекта из нескольких отдельных позиций

- Комплектация

  формирование набора оборудования и материалов в соответствии с заказом или проектом

- Коэффициент полезной площади (КПП)

  отношение полезной (используемой под хранение/деятельность) площади к общей площади помещения или объекта

- Кросс-докинг

  логистическая операция, при которой грузы принимаются на складе и тут же, без размещения в зоне длительного хранения, перегружаются для дальнейшей отправки конечному получателю

- Кросс-докинг/транзит

  схема, при которой товары перегружаются с одного транспортного средства на другое без длительного хранения    

{% endlist %}

**Л**

{% list tabs accordion %}

- Логистические данные продукта

  весогабаритные характеристики товара в упаковке

{% endlist %}

**М**

{% list tabs accordion %}

- Маркировка

  нанесение специальных обозначений (штрих-кодов, стикеров, бирок) для идентификации и отслеживания оборудования

- Минимальная отгрузочная единица

  наименьшая самостоятельная единица товара, которая может быть отгружена, например, штука, пачка, коробка или паллета

{% endlist %}

**Н**

{% list tabs accordion %}

- Напольное хранение

  метод организации складского пространства, при котором размещение товаров осуществляется непосредственно на полу склада, чаще всего на поддонах (паллетах), без использования стеллажей

- Номенклатура

  сущность в системе, по которой считают остатки на складе, проводят списание и поступление

{% endlist %}

**О**

{% list tabs accordion %}

- Основные средства склада

  долгосрочные материальные активы склада, которые используются в его деятельности (производстве, оказании услуг, управлении) более 12 месяцев и способны приносить экономическую выгоду

{% endlist %}


**П**

{% list tabs accordion %}

- Паллета

  стандартизированная деревянная, пластиковая или металлическая платформа с выемками для погрузчика, предназначенная для формирования единой грузовой единицы при хранении и транспортировке товаров

- Передача оборудования

  выдача оборудования клиенту или подрядчику с подтверждением факта передачи документально

- Перемещение

  процесс организации доставки или складская операция, которая отражает перемещение каких-либо материальных предметов, рекламных материалов из одной точки в другую по оптимальному маршруту

- Пересорт (пересортица)

  выявление недостачи одного товара и/или излишек другого товара

- Пикинг коробочный

  отбор товаров на складе по коробкам для дальнейшей комплектации или отгрузки

- Плановая инвентаризация

  обязательная ежегодная инвентаризация, проводимая по заранее утвержденному графику

- Погрузо-разгрузочные работы (ПРР)

  операции по перемещению оборудования при приёмке, отгрузке или транспортировке

- Полезная площадь

  часть общей площади, используемая для размещения или хранения товаров

- Поступление

  операция приема оборудования на баланс склада

- Приёмка оборудования

  формальный процесс получения оборудования от перевозчика или подрядчика с проверкой количества, качества и соответствия документации

- Приказ на отгрузку Москва

  документ, на основании которого склад подготавливает материалы для заявки на отгрузку/прием контрагента

- Присвоение категории качества

    определение состояния оборудования и фиксация этой информации в системе

- Проверка оборудования по качеству

  осмотр оборудования на наличие дефектов, повреждений или брака

- Проверка оборудования по количеству

  сверка фактического числа единиц оборудования с данными сопроводительных документов

{% endlist %}

**Р**

{% list tabs accordion %}

- Разукомплектация

  операция по разделению единого объекта (комплекта товаров, основного средства) на отдельные составляющие, являющаяся обратным процессом комплектации

- Расходные материалы

  товары, которые предназначены для однократного или ограниченного по времени использования в процессе производства, обслуживания или выполнения различных операций

- Ремонт

  операция, отражающая изменение состояния оборудования, восстановление его работоспособности и устранение дефектов

- Рохля

  гидравлическая тележка, используемая для ручной транспортировки поддонов и других грузов на складах, производстве и т.п.

{% endlist %}

**С**

{% list tabs accordion %}

- Состояние новое

  оборудование, поступившее на склад, но не бывшее в эксплуатации

- Состояние оборудования

  характеристика оборудования в разрезе истории его эксплуатации

- Состояние ремонт

  оборудование, бывшее в эксплуатации в торговой точке и поступившее на склад

- Состояние старое

  оборудование, бывшее в состоянии «Ремонт» и прошедшее процедуру восстановления

- Состояние утиль

  неремонтопригодное оборудование, которое необходимо утилизировать

- Списание

  операция списания оборудования с баланса склада

- Стеллажи

  складское оборудование для хранения товаров в несколько ярусов

- Стеллажное хранение

  метод организации складского пространства, при котором товары размещаются на многоярусных конструкциях (стеллажах)

{% endlist %}

**Т**

{% list tabs accordion %}

- Топология склада

  представление внутреннего пространства склада, разделенного на функциональные зоны и содержащего технологические элементы

{% endlist %}

**У**

{% list tabs accordion %}

- Упаковка

  подготовка оборудования к хранению или транспортировке: обёртывание, закрепление, защита от повреждений

- Утилизация

  вывоз и переработка (или уничтожение) оборудования, не подлежащего дальнейшему использованию

{% endlist %}


**Ф**

{% list tabs accordion %}

- Формирование паллет к отгрузке

  сбор и размещение коробок или товаров на паллетах с учётом направления поставки и грузополучателя

{% endlist %}


**Х**

{% list tabs accordion %}

- Хранение

  размещение оборудования на складе с обеспечением сохранности и учёта

{% endlist %}

**Ц**

{% list tabs accordion %}

- Центральный склад (ЦС)

  склад клиента, с которого поступает новое оборудование

- Циклический учёт

  постоянный процесс проверки фактического наличия и состояния имущества организации и её обязательств путем сверки с данными бухгалтерского учёта с установленной регулярностью

{% endlist %}

**Ш**

{% list tabs accordion %}

- Штабелер

  подъёмно-транспортное оборудование для подъёма, перемещения и укладки грузов, чаще всего паллетированных, в несколько ярусов

{% endlist %}


**Я**

{% list tabs accordion %}

- Ячейка

  место хранения на складе в случае наличия адресной системы хранения

{% endlist %}


## Транспортная логистика

**F**

{% list tabs accordion %}

- Full Truck Load / Прямая Машина (FTL/ПМ)

    заказ целого транспортного средства под один заказ. Быстрее, дороже LTL, минимизирует риски повреждений

{% endlist %}


**L**

{% list tabs accordion %}

- Less Than Truckload / Сборный груз (LTL/СГ)

  перевозка партии груза совместно с грузами других заказчиков. Дешевле FTL, дольше, выше риск задержек/повреждений

{% endlist %}

**T**

{% list tabs accordion %}

- Transport Integration Core (TIC)

  система управления транспортной логистикой

{% endlist %}

**Г**

{% list tabs accordion %}

- Грузополучатель

  контрагент или доверенное лицо контрагента, принимающее груз, в конечном адресе получения груза

{% endlist %}

**О**

{% list tabs accordion %}

- Объемный вес

  расчетный вес груза, основанный на габаритах и применяемом коэффициенте. Используется при определении стоимости перевозки. Обладает преимуществом при условии превышения физического веса

{% endlist %}

**П**

{% list tabs accordion %}

- Платный вес

  значение, используемое для расчета стоимости доставки груза, которое представляет собой наибольшее из двух показателей: фактического веса груза (веса брутто) и его объемного веса. Он рассчитывается, чтобы перевозчики могли максимально эффективно использовать пространство в транспортном средстве, взимая плату за груз, занимающий больше места, даже если его фактический вес невелик

- Подтверждение грузополучателей

  процедура назначения корректного контрагента и грузополучателя

- Последняя миля

  финальный этап доставки груза от склада/хаба непосредственно до места доставки и/или проведения работ

{% endlist %}

**Р**

{% list tabs accordion %}

- Региональная рассылка (РР)

  комплекс мероприятий по отправке грузов, включающая в себя все этапы от поступления товара на склад до его выдачи получателю или другому звену цепи поставок

{% endlist %}

**Т**

{% list tabs accordion %}

- Трейс

  документированный проблемный случай с перевозкой или доставкой груза

- Трек-номер / экспедиторская расписка (ЭР)

  номер экспедиторской расписки, накладная, по которой можно отслеживать движение груза

{% endlist %}

**Ф**

{% list tabs accordion %}

- Физический вес

  масса груза, измеренная на весах. Он отражает фактическую массу отправления с учетом всей упаковки и тары

{% endlist %}

**Э**

{% list tabs accordion %}

- Экспресс-доставка

  срочная доставка малогабаритных (обычно, но не всегда) отправлений (документы, малые формы ПОСМ) курьерскими службами, часто с использованием авиатранспорта. Высокая скорость, высокая стоимость

{% endlist %}
