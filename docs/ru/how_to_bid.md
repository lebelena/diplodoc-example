# Как создать заявку

Заявка — это документ, который необходим для фиксации поступления, списания и инвентаризации оборудования на склад. 

{% list tabs %}

- Поступление

  На вкладке «Основная информация» указываются
  - планируемые даты начала и завершения
  - нужно ли хранение: отсутсвует, длительное, [сквозное складирование](*key_storage)
  - комментарий: будет виден для исполнителей и руководителя склада, можно при необходимсоти прикрепить файлы для уточнения комментария
  - контрагент, который будет выполнять работу по заявке
  - договор, в рамках которого будут выполняться работа

  Вкладка с номенклатурой будет недоступна, пока не будет заполнена основная информация.

  На вкладке «Номенклатура» указывается та номенклатура//оборудование?, с которой будут работать в заявке.

- Списание

  На вкладке «Основная информация» указываются
  - планируемые даты начала и завершения
  - причина списания: поставка, брак, бой, истечение срока годности
  - комментарий: будет виден для исполнителей и руководителя склада, можно при необходимсоти прикрепить файлы для уточнения комментария
  - контрагент, который будет выполнять работу по заявке
  - договор, в рамках которого будут выполняться работа
  
  Вкладка с номенклатурой будет недоступна, пока не будет заполнена основная информация.

  На вкладке «Номенклатура» указывается та номенклатура//оборудование?, с которой будут работать в заявке. Выбор осуществляется из остатков

- Инвентаризация

  На вкладке «Основная информация» указываются
  - планируемые даты начала и завершения
  - комментарий: будет виден для исполнителей и руководителя склада, можно при необходимсоти прикрепить файлы для уточнения комментария
  - контрагент, который будет выполнять работу по заявке
  - договор, в рамках которого будут выполняться работа
  
  Вкладка с номенклатурой будет недоступна, пока не будет заполнена основная информация.

  На вкладке «Номенклатура» указывается та номенклатура//оборудование?, с которой будут работать в заявке.

{% endlist %}







[*key_storage]: Сквозное складирование — логистическая схема, по которой грузы принимаются на складе и тут же, без размещения в зоне длительного хранения, перегружаются для дальнейшей отправки конечному получателю.
