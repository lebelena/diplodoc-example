# Как создать заявку

Заявка — это документ, который необходим для работы с номенклатурой внутри компании. 

У каждой заявки есть вкладки с основной информацией и информацией о номенклатуре.

Выберите тот тип заявки, который хотите создать: поступление, списание или  инвентаризация и заполните данные в соответствии с выбранным типом. 
  
{% note warning %}

Вкладка с номенклатурой будет недоступна, пока не будет заполнена основная информация.

{% endnote %}

{% list tabs %}

- Поступление
  
  * На вкладке «Основное» указываются
    - планируемые даты начала и завершения,
    - хранение для номенклатуры: отсутствует, длительное, [кросс-докинг](*key_storage),
    - комментарии: будет виден для исполнителей и руководителя склада, при необходимости можно прикрепить файлы для конкретизации, 
    - контрагент, который будет выполнять работу по заявке,
    - договор, в рамках которого будут выполняться работа на складе.

    ![_screen_type_task](_images/_screen_type_task.png){.center width=1200}

  * На вкладке «Номенклатура» введите в поиске наименование, артикул или SAP-код необходимой номенклатуры и по команде «Добавить» откройте карточку номенклатуры. Заполните все поля и по команде «Сохранить» добавьте номенклатуру в заявку.  
   
    ![_screen_task_nomenc](_images/_screen_task_nomenc.png){.center width=1200}

    Когда вся номенклатура добавлена, создание заявки завершается по команде «Создать».

    ![_screen_task_list_nomenc](_images/_screen_task_list_nomenc.png){.center width=600}

- Списание

  На вкладке «Основная информация» указываются
  - планируемые даты начала и завершения
  - причина списания: поставка, брак, бой, истечение срока годности
  - комментарий: будет виден для исполнителей и руководителя склада, при необходимости прикрепите файлы для уточнения комментария
  - контрагент, который будет выполнять работу по заявке
  - договор, в рамках которого будут выполняться работа
  
  Вкладка с номенклатурой будет недоступна, пока не будет заполнена основная информация.

  На вкладке «Номенклатура» указывается та номенклатура//оборудование?, с которой будут работать в заявке. По каждой номенклатуре указано то количество, которое хранится на складе. Выбор происходит из имеющихся остатков.
