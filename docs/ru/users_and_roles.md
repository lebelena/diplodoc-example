# Пользователи и роли

Все пользователи системы называются **контрагентами**. 
У разных контрагентов — разный уровень доступа. Уровень доступа определяется занимаемой должностью, указанной в договоре при регистрации. 

Зарегистрироваться самостоятельно нельзя. Пользователь получает логин и пароль по  элетронной почте, после того, когда HR-менеджер зарегистрировал его в системе.
 
На данный момент в системе есть роли:
* директора,
* менеджера,
* руководителя проекта,
* HR-менеджера,
* начальника склада,
* кладовщика,
* координатора,
* специалиста ТМЦ.
