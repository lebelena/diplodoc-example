# Как создать поручение

Поручение всегда создается в рамках договора.

Чтобы открыть договор, перейдите в подсистему «Справочники» → «Контрагенты» и найдите блок «Договоры». 

{% note info %}

Если у договора есть поручения, то в верхнем углу отобразится счетчик активных поручений к общему числу.

{% endnote %} 

![_screen_assign_docs](_images/_screen_assign_docs.png){.center width=1200}

## Шаг 1. Заполнение информации о поручении

Выберете договор, в рамках которого хотите создать поручение, и по команде «Открыть» → «Добавить поручение» откройте форму создания поручения.

Необходимо указать:
- услуги: выбрать можно только среди тех, которые указаны в договоре;
- срок действия поручения;
- внешний номер поручения: заполняется, если поручение сформировано в сторонней системе;
- номенклатура: указывается та, над которой будет производиться работа в рамках поручения. 

![_screen_create_assign](_images/_screen_create_assign.png){.center width=1200}

По команде «Далее» поручение закрывается и отправляется генеральному директору на согласование. До тех пор, пока генеральный директор не согласует поручение, оно будет недоступно. 

## Шаг 2. Согласование поручения

После создания поручения генеральный директор в списке мероприятий видит созданное поручение. 

В карточке поручения видна вся ранее внесенная информация и можно либо подтвердить, либо отклонить поручение.

![_screen_card_assign_gen](_images/_screen_card_assign_gen.png){.center width=400}

**При отклонении:**
1. генеральный директор указывает причину отклонения и может прикрепить файлы для уточнения причины отказа;

![_screen_card_assign_gen](_images/_screen_assign_cancelled.png){.center width=400}

2. пользователь, создавший поручение, видит отклонненое поручение в списке своих мероприятий, откуда можно посмотреть причину отклонения и доработать поручение.  

{% note warning %}

Поручение может быть отклонено несколько раз. Все изменения хранятся в системе и их можно посмотреть переключаясь между событиями в левой части экрана. 

{% endnote %}  

![_screen_gen_cancelled](_images/_screen_gen_cancelled.png){.center width=1200}

**При подтверждении:**
пользователь, создавший поручение, видит подтвержденное поручение в списке своих мероприятий и может загрузить документы (шаг 3).

## Шаг 3. Загрузка документов

Чтобы завершить создание поручения, необходимо прикрепить файл с поручением в формате pdf по команде «Прикрепить файл» или перетащите его в область загрузки.   

![_screen_load_file_assign](_images/_screen_load_file_assign.png){.center width=1200}

Создание поручения завершается по команде «Создать». 
Система выведет сообщение об успешном создании поручения с указанием внутреннего номера.

![_screen_load_file_assign](_images/_screen_load_file_assign.png){.center width=400}

## Просмотр информации о поручениях

Информацию о поручениях можно просмотреть из карточки контрагента в конкретном договоре, если в рамках договора уже были созданы поручения: в карточке договора будет отдельная вкладка «Поручения».

![_screen_assigndocs](_images/_screen_assigndocs.png){.center width=800}

Отсюда можно просмотреть и отредактировать, созданное поручение, а также создать заказ на основании поручения по команде «Перейти к заказам».
