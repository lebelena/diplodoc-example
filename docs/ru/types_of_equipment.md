# Виды оборудования

**Вид оборудования** - это справочник, который позволяет познакомиться со всеми существующими видами оборудования. 

 ![_screen_equip_list](_images/equip_list.png){.center width=1200}

При клике по карточке можно увидеть оборудование, которое относится к выбранному виду, и посмотреть подробную информацию также кликнув на нужную строку. 

 ![_screen_equip_list_2](_images/equip_list_spec.png){.center width=1200}
