# Типы оборудования

{% note info %}

**[Тип оборудования][2]** — сущность, необходимая для объединения разных видов оборудования с общими признаками в одну группу. 

**[Вид оборудования][1]** — справочник с информацией об объектах, над которыми выполняются различные операции. 

{% endnote %}

Данные о типах и зависимых видах оборудования содержатся в [подсистеме][3] «Справочники» → «Типы оборудования».
- **Тип оборудования** отображается графически, под каждым изображением указано наименование типа.
- Для перехода к **виду оборудования** необходимо кликнуть по соответствующему типу, в списке найти нужное наименование и выбрать его — откроется [форма][4], содержащая в себе всю информацию.

![_screen_equip_list_2](_images/equip_list_spec.png){.center width=1200}

{% note info %}

**Если известно только название вида, воспользуйтесь поиском.** Система определит тип, к которому может относиться искомый вид оборудования, и покажет все [карточки][5], где встречается указанное наименование.

![_screen_equip_list_2](_images/equip_search_list.png){.center width=1200}

{% endnote %}

## Страница вида оборудования

Страница(форма) вида оборудования содержит:
- список действий, которые необходимо осуществить, чтобы его установить(1);
- справочную информацию и связанную [номенклатуру][6](2). 

![_screen_equip_list_2](_images/_screen_equip_types.png){.center width=1200}

### Список действий над оборудованием

У каждого действия есть 3 [иконки][7] с информацией о:
- требованиях к охране труда;
- составе команды;
- ожидавемом времени выполнения.
При клике по действию разворачивается вся информация, которая суммарно отображена в иконках.  

![_screen_equip_list_2](_images/equip_hide_step.png){.center width=1200}

### Справочная информация

**Справочная информация** содержит сведения о материале, способе крепления, габаритах оборудования и категорию сложности работ. Также в зависимости от характеристик оборудования автоматически выбирается вид транспорта и сопутствующая информация. 

![_screen_equip_list_2](_images/equip_type_transp.png){.center width=1200}

### Номенклатура вида оборудования

**Вкладка «Номенклатура»** содержит список тех номенклатур, для которых был выбран данный вид оборудования. 
Отсюда можно перейти к карточке конкретной номенклатуры (переход происходит при клике по наименованию номенклатуры) и посмотреть заказ, в котором она была использована (переход по номеру заказа).

![_screen_equip_list_2](_images/equip_nomenc.png){.center width=1200}


{% note info %}

**Руководитель РПУ может изменить всю справочную информацию**, кроме автоматически рассчитываемой. Для остальных пользователей доступен только просмотр. 

{% endnote %}
