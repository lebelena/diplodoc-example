# Список заказов

Заказы, приказы и наряды можно посмотреть в подсистеме «Список заказов».

В списке для каждого заказа отображены:
- номер и наименование заказа — по значениям этих полей можно искать нужный заказ через поисковую строку в списке; 
- заказачик, менеджер, дата создания, статус — можно отфильтровать список по одному или нескольким параметрам одновременно;  
- поручение — показывает связь с договором; 
- состояние — отражает проценты выполнения разных типов задач по всему заказу:
  * <span style="color: #F9AF49;">100 %</span> — задачи склада;
  * <span style="color: #08BCC9;">88 %</span> — региональная рассылка;
  * <span style="color: #1EC3A5;">88 %</span> — наряды; 
- финансовый показатель — сумма всех утвержденных работ по заказу. 

![list_of_orders](_images/_screen_list_of_orders_mounth.png){.center width=1200}

При клике по любой строке открывается [диаграмма Ганта](*key_gant) на текущий месяц.

![gant_mounth](_images/_screen_gant_mounth.png){.center width=1200}

По-прежнему остаются активными фильтры вверху диаграммы. 
По горизонтали отображены числа текущего месяца, вертикально — задачи по одной заявке. Красной вертикальной линией отмечено текущее число. 
В правой части окна можно выбрать
- режим отображения: месяц, неделя, день — можно переключаться между режимами, зажав Alt на клавиатуре и прокручивая колесо мыши вперед или назад в зависимости от выбранного режима отображения;
- месяц, по которому строится диаграмма — можно переключаться между месяцами, зажав Shift на клавиатуре и прокручивая колесо мыши вперед или назад в зависимости от того месяца, который требуется открыть;
- заказ, по которому отображаются задачи.

Задачи на диаграмме группируются по пространству списания из региональной рассылки: если пространства списания разные или одинаковые, но с разными датами списания, то задачи будут разнесены на несколько групп. 
*На скриншоте ниже можно выделить 2 группы задач.* 

![gant_2_groups](_images/_screen_gant_2_groups.png){.center width=1200}

Можно кликнуть по любой задаче и увидеть всплывающее окно со статусом, номером, датами и наименованием пространства. 

![info_task](_images/_screen_info_task.png){.center width=400}

У части задач есть цифры в правом верхнем углу:
- для задач типа «региональная рассылка» — последующие задачи на поступление
![task_reg_sender](_images/_screen_reg_sender_task.png){.center width=400}

- для задач типа «поступление» — количество списаний, которые необходимо выполнить после
![task_from_1_to_3](_images/_screen_task_from_1_to_3.png){.center width=400}

- для задач типа «наряд» — это количество подзадач: по клику скрытые задачи развернутся вниз на диаграмме   
![task_1_in_6](_images/_screen_task_1_6.png){.center width=400}

**В рамках недельного отображения** по горизонтали отображается время, по вертикали — дни, относящиеся к выбранной неделе с понедельника по воскресенье. 

Красной линией отмечено текущее время.

1 строка = 1 день. В рамках одного дня отображаются все задачи на день. 
Если задача идет несколько дней подряд, каждый новый день отображается [хедер](*key_task) задачи. 

Если все задачи не помещаются, то такие задачи подсчитываются и скрываются под командой с числом невошедших задач. По клику список скрытых задач отображается справа. 

![_screen_week_hidden_task](_images/_screen_week_hidden_task.png){.center width=1200}





  






[*key_gant]: Диаграмма Ганта — это график работ по проекту. Диаграмма состоит из двух осей: вертикальной со списком задач и горизонтальной со сроками.

[*key_task]: Хедер задачи — это плашка в левом верхнем углу у задачи на диаграмме Ганта. Отражается номер и тип задачи.
